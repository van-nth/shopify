# Shopify Dawn Theme Extension - Product Grid/List View

![Screen recording](./assets/product-grid-list-view.gif)

### Description:

    - A custom product grid section with View as grid layout or list layout
    - Customizable in section settings
    - Enabled on: [collection] template only

## Installation

Copy and paste [product-grid-list-view.liquid](./sections/product-grid-list-view.liquid/) into `sections` folder in your Theme

## Usage

1. Go to Theme editor > Select **collection** template
2. Add `product-grid-list-view` section to the template
3. Hit **Save**
